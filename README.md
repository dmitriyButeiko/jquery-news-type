# jquery-news-tape
## Overview
Simple jQuery News Tape
## Initialization
---
````html
<div id="newsTape">News string, which will be in tape</div>
````
````javascript

var speed = 40000; // default speed equals 20000 milliseconds
$("#newsTape").newsTape(speed);
````
